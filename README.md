## 감정분석기능
기본적으로 저렴한 llm의 api를 이용하여 적절한 프롬프트로 문장별 감정분석을 진행  
사전에 여러개의 감정을 미리 지정해놓고, 해당하는 감정중에서 적절한 것을 선택할 것이다.

서버와 구분되는 별도의 모듈로 구현하는것으로 생각중이다.

llm 호출에 따른 비용이 존재하므로 비용 최소화를 위해 아래의 최적화 방식을 고려중이다.
